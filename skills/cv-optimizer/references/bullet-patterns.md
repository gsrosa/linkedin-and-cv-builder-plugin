# Bullet pattern classification

Use after [bullet-analyze.md](bullet-analyze.md) to label **how** to rewrite. See [bullet-rewrite.md](bullet-rewrite.md) for rewrite rules and output template.

## Objective

Classify the bullet into one or more rewrite patterns.

## Available patterns

1. **Vague → Specific**
2. **Responsibility → Impact**
3. **Participation → Ownership**
4. **Feature → Outcome**
5. **Tool → Decision**
6. **Implementation → Problem-Solution**
7. **Basic → Scale/Complexity**
8. **Task → System Thinking**
9. **Generic → Trade-off Explanation**
10. **Weak → Business Impact**

## Instructions

- A bullet can match multiple patterns
- Prioritize the **highest-impact** transformation
- Do **not** over-apply patterns

## Output (classification phase)

- Selected pattern(s)
- Why they apply

**Then:** [bullet-rewrite.md](bullet-rewrite.md)
