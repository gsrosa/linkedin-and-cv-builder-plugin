# Post critic

Evaluate and improve post quality **before** final output.

## Questions

- Is this generic?
- Could anyone have written this?
- Is there a **real** insight (non-obvious, grounded)?
- Is the hook strong enough to stop scrolling?
- Is it too long? (Cut ruthlessly — LinkedIn rewards density)
- Any banned phrases or corporate tone? ([post-creation-engine.md](post-creation-engine.md))

## If weak

- Rewrite the hook first
- Add specificity (numbers, constraints, what broke, what you tried)
- Inject a stronger opinion or a sharper failure story
- Remove filler and duplicate ideas

## Pre-delivery checklist (merge)

1. Could this have been written by anyone? → More specific.
2. Does it sound like someone with **real** shipping experience?
3. Is the hook strong enough? → Rewrite if soft.
4. Is there filler? → Cut it.
5. Does the close earn engagement (specific question or punchy lesson) — not "Agree?" or "Thoughts?"

## Pass / fail

- **Pass:** Specific, voice, insight, scannable.
- **Fail:** Generic, soft hook, advice anyone could post — return to [post-creation-engine.md](post-creation-engine.md) and [hook-generator.md](hook-generator.md).
